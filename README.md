## 🚀 About

**OtakuCloud Stream** is a modern anime streaming app interface built with a premium UI inspired by platforms like Netflix & Crunchyroll.

> ⚠️ This project currently does **not include a backend connection**  
> Streaming requires API integration (Consumet / custom backend)

---

## 📱 Download APK

👉 https://knave.in  

### 📥 Steps:
1. Open the site  
2. Click the **first blue "Knave" button**  
3. Download & install the APK
